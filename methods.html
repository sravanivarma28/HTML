Methods

  1. A method is a block of code which only runs when it is called.
  2. You can pass data,known as parameters,into a method.
  3. Methods are used to perform certain actions, and they are also known as functions.
  4. Why use methods?To reuse code : define the code once,and use it many times.
  5. static means that the method belongs to the Myclass class and not an object of the Myclass class.You will learn more about objects and how to access methods through objects later.
  6. void maens that this mrthod does not have a return value.Ypu will learn about retutn values later in this Chapter.
